API Reference

GET/api/status
Return all recorded backup entries as a JSON array.
DELETE/api/status
Clear all recorded backup entries.
GET/api/configs
Return the list of loaded config blocks (name, sources, destinations, cron spec, next run).
GET/api/running
Returns {"running": true|false} — whether a backup run is currently in progress.
POST/api/run
Trigger an immediate backup run.
Body: {"index": -1} — -1 runs all configs, ≥0 runs the config at that index.
Returns 202 Accepted or 409 Conflict if already running.
GET/api/config
Read a config file.
?file=0 — zero-based file index (default 0).
?list=1 — list all registered config files as JSON instead.
POST/api/config
Overwrite a config file with the plain-text request body.
?file=0 — zero-based file index (default 0).
